------------------------------------------------
ESTRATÉGIAS DE BUSCA INFORMADAS E NÃO INFORMADAS
------------------------------------------------



INSTALAÇAO/EXECUÇAO
-------------------
	
	Caso pretenda pode compilar o ficheiro SI.java.
	Na linha de comandos e no directorio para onde extrair os ficheiros executar o comando: 
	
	javac SI.java 


	No entanto o programa já vem pre-compilado.
	Na linha de comandos e no directorio para onde extrair os ficheiros executar o comando: 
	
	java SI



COMO USAR
---------

	1.O programa irá automaticamente executar os testes de paridade no tabuleiro 1 e 2 em relação ao final e apresentar o resultado dos testes.

	2.Após isso irá prosseguir para o menu de escolha da estratégia a utilizar para resolver o unico tabuleiro resolvivel -> 2.
	As 5 opções de pesquisa serão apresentadas no ecrã, bastando digitar o numero da pretendida e carregar em ENTER.
	A pesquisa será executada e os resultados serão apresentados. Após isso, o programa terminará.

	3.Para novos resultados, executar novamente o programa.


ficheiros compilados utilizando linha de comandos windows 8 e java 1.7.0_13
